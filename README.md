# RecipeAI

## Project Overview
RecipeAI is an intelligent recipe management application designed to help users discover, create, and manage recipes with ease. The application leverages AI technology to suggest personalized recipes based on user preferences and dietary restrictions.

## Features
- **Recipe Suggestions:** Get suggestions for recipes based on ingredients you have.
- **User Preferences:** Customize your experience by setting dietary preferences and restrictions.
- **Add/Manage Recipes:** Users can add their own recipes or edit existing ones.
- **Search Functionality:** Easily search for recipes by keywords or ingredients.
- **Rating System:** Users can rate recipes and leave reviews to assist others in choosing the best meals.

## Tech Stack
- **Frontend:** React, Redux
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI Integration:** Custom AI models for recipe suggestions

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/SarathNK-code/recipeai.git
   ```
2. Navigate to the project directory:
   ```
   cd recipeai
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Usage
Once the application is running, navigate to `http://localhost:3000` in your web browser. From there, you can explore recipes, add your own, and customize your user preferences.
